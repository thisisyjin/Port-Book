about

React에서의 API 요청 및 라우팅 에 대해 연습하고자 제작한 영화 정보 서비스 프로젝트.

영화 정보 API를 비동기 요청하여 렌더링하고, 데이터 (연도, 별점, 런타임, 줄거리)를 렌더링.
react-router 라우팅과 params를 이용하였고, module.css로 스타일링. (+ 슬라이드 구현)

es6의 async/await를 이용한 비동기 요청을 구현.
React Hooks인 useCallback 등을 이용한 성능 최적화 진행.

Stack

Axios 를 통한 (API) 비동기 요청, 상태관리 라이브러리인 Recoil, Prop-types 등.
